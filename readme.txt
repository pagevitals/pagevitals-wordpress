Integrates PageVitals Field Testing (RUM) into your WordPress site.

== Description ==

This plugin allows you to easily integrate PageVitals Field Testing (RUM) into your WordPress site. It provides options to add the PageVitals script to all pages or specific pages, and can also add necessary Content Security Policy headers.

== Installation ==

1. Upload the `pagevitals-wordpress` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > PageVitals RUM to configure the plugin

== Frequently Asked Questions ==

= Where can I find my Website ID? =

You can find your Website ID in your PageVitals dashboard. Click Settings and you'll find the ID in a blue box next to the section "Website Settings". If you don't have an account, sign up at https://www.pagevitals.com/
